## Interactions

### Drag-to-Reorder

1. Click and hold grip handle
2. Drag vertically
3. Other cards animate to make space
4. Release to drop in new position
5. Positions update automatically

### Alternative Reordering

1. Click up/down arrow buttons
2. Role swaps with adjacent role
3. Smooth animation
4. Positions update automatically

### Expand/Collapse

1. Click chevron button
2. Card content animates height
3. Content fades in/out
4. Other cards shift smoothly

### Edit Role

1. Click "Edit" button
2. Dialog opens with animation
3. Form pre-filled with role data
4. Save updates role
5. Dialog closes with animation

### Delete Role

1. Click "Delete Role" (only if allowed)
2. Confirmation appears
3. Click "Yes, Delete" to confirm
4. Role removed with animation
5. Remaining roles reorder

### Create Role

1. Click "+ Create Role" button
2. Dialog opens with empty form
3. Fill in details
4. Click "Create Role"
5. New role appears at bottom
